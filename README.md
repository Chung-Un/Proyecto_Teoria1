Proyecto Gestor de Presupuestos Teoría de Base de Datos 1 

Descripción general
El presente proyecto permite la gestión de presupuestos personales de un usuario; facilitando así la planificación de sus finanzas. 
Tomando en cuenta sus obligaciones personales mensuales, metas de ahorro y observaciones adicionales con respecto a estas. 

Objetivos
Aplicar los conocimientos de Fundamentos de Sistemas de Bases de Datos mediante el diseño, implementación y despliegue de una solución completa de gestión financiera personal.

Características principales
-Gestión de usuarios
-Planificación presupuestal
-Registro de transacciones
-Gestión de obligaciones fijas
-Analítica y reportería

Tecnologías usadas
-Microsoft SQL Server
-C#
-Entitity framework (LINQ)
-Además incluirá un frontend simplístico.
