Proyecto Gestor de Presupuestos Teoría de Base de Datos 1 

->El presente proyecto permite la gestión de presupuestos personales de un usuario; facilitando así la planificación de sus finanzas. 
Tomando en cuenta sus obligaciones personales mensuales, metas de ahorro y observaciones adicionales con respecto a estas. El proyecto
estará creado en C#, utilizando Microsoft SQL server; se implementará un UI simplístico utilizando las librerias que C# ofrece.
